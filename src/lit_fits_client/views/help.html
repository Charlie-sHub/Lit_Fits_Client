The Register window works to register a new user, meaning you can't use the same username as one already signed up. Make sure the email is a proper email or otherwise the program won't let you sign up. You must also re-type the password to make sure it has no errors.

Use the buttons Undo and Redo to erase everything and write it again respectively. Cancel closes the current window and opens the login window again. Register submits the data entered to the server. Help opens this window. Happy mode makes your day brighter.

The Register window has the following hot-keys
  • Alt + r : Attempts to submit the data entered to register
  • Alt + c : Cancels the whole procedure and opens the login vindow
  • F1 : Opens the help window, the one you're seeing right now
